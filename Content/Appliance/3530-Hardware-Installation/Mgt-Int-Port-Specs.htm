Management Interface Port Specifications

The management network interfaces (eth1 and eth2) found on the rear of the Solace PubSub+ 3530 use single RJ-45, Automatic Medium-Dependent Interface Crossover (Auto-MDIX) ports with auto-negotiation for the following bandwidths. Connect to these ports with CAT-5e unshielded twisted pair cable.

Appliance Management Interface Port Specification
CHS-3530AC-01-A 10/100/1000 base T
CHS-3530AC-02-A 10/100/1000 base T
CHS-3530AC-03-A 1000 base T
