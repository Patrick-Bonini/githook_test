Management Interface Port Specifications

The management network interfaces (that is, eth1 and eth2) found on the rear of the Solace PubSub+ 3260 are single RJ-45, Automatic Medium-Dependent Interface Crossover (Auto-MDIX) ports with auto-negotiation for 10/100/1000 base T.

Connect to these ports with CAT-5 unshielded twisted pair cable.
