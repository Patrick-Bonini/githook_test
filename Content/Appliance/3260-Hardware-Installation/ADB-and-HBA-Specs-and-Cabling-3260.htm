ADB and HBA Specifications and Cabling

This section provides details on optical port cabling specifications for the Assured Delivery Blade (ADB) and associated Host Bus Adapter (HBA).

SFP Modules

The ADB and HBA interfaces provide Fiber Channel (FC) connections using an LC physical connector () for optical fiber connections using shortwave laser optics. The ADB and HBA SFP modules are field-replaceable and hot-swappable.

  • ADB Interface—The interfaces on the ADB are two 4 Gbps FC SFP optical module ports. The ADB optical ports run a Solace proprietary protocol.
  • HBA Interface—The interfaces on the HBA-0204FC are two 4 Gbps FC SFP optical module ports, while the interfaces on the HBA-0208FC are two 8 Gbps SFP optical module ports. The HBA optical ports run standard fiber channel protocols.

ADB Faceplate LEDs

Faceplate LEDs on the ADB indicate the link status for the two ADB SFP module ports, and the read/write status for the Flash Memory Card(s). One green and yellow status LED is used for each SFP module port, and one red status LED is used for each internal Flash Memory Card (one for the ADB-000000-02, two for the ADB-042-10M-01-A).

The green port link status LED indicates the state of the SFP physical link:

  • Off—Either no link to the SFP module has been established, the module is not receiving power, or the cable connection is not secure.
  • On—The link to the SFP module has been established, the module is receiving power, and the cable connection is secure.

    If the appliance is reloaded when no SFP module is installed in the mate-link port, the LED behavior for that port will be undefined. However, the port LEDs will properly display the current status once an SFP module is installed and the hardware drivers identify it.

The yellow activity status LED indicates whether there is data activity for the SFP modules:

  • Off—The SFP module is not receiving or transmitting messages (including network acknowledgments).
  • ON—The SFP module is receiving and transmitting messages (including network acknowledgments).

The red memory card status LED indicates the state of the flash memory card:

  • Off—No reading or writing of spooled messages from the flash memory card.
  • On—Initialization, read, or write operations are occurring for the flash memory card.

    The blade carrier containing the ADB must not be removed from the chassis when the Red LED is On.

Connecting to ADB and HBA SFP Devices

The SFP modules on ADBs and associated HBAs use LC connectors for Fiber Channel connections using shortwave laser optics over multimode optical fiber (MMF) cable.

The ADB mate link uses a proprietary cell-based protocol. Refer to Optical Loss Budget and Operating Distances for the associated optical loss budget and operating distance.

It is recommended that the maximum difference in length between the two cables required for the mate link does not exceed 20 m.

The HBA-0204FC operates according to the 4 Gbps Fiber Channel specification, and the HBA-0208FC operates according to the 8 Gbps Fiber Channel specification. Refer to Optical Loss Budget and Operating Distances for the associated optical loss budget and operating distance.
