Management Interface Port Specifications

The management network interfaces (eth1 and eth2) found on the rear of the 3560 appliance use single RJ-45, Automatic Medium-Dependent Interface Crossover (Auto-MDIX) ports with auto-negotiation for the following bandwidths:

Appliance Management Interface Port Specification
CHS-3560AC-01-A 10/100/1000 base T
CHS-3560AC-03-A 10/100/1000 base T
CHS-3560AC-05-A 1000 base T
CHS-3560AC-06-A 10/100/1000 base T

  • CAT-5e unshielded twisted pair cable (minimum requirement).
