Redundant Power Supply Settings

Solace PubSub+ appliances use redundant AC hot-swappable power supplies.

  • PubSub+ 3530s and 3560s use two redundant AC hot-swappable power supplies—one installed power supply meets the system requirements, and the second installed power supply provides redundancy, load sharing, and increased system availability.
  • PubSub+ 3260s use three or four redundant AC hot-swappable power supplies—two installed power supplies meet the system requirements. The third and fourth installed power supplies provide redundancy, load sharing, and increased system availability, and they can be removed (or inserted) without affecting system operation.

For a Solace PubSub+ 3260 you can configure the redundant power supply parameter that raises syslog event message when the number of active power supplies detected on the appliance is less than the configured power redundancy parameter.

To configure the redundant power supply setting on a Solace PubSub+ 3260, enter the following CONFIG command:

solace(configure)# hardware power-redundancy <type>

Where:

type specifies the redundant power supply configuration to use for the appliance. Valid values for a PubSub+ 3260 are:

  • 2+1 (requires at least three active power supplies)
  • 2+2 (requires four active power supplies). This is the default for a Solace PubSub+ 3260.
  • 1+1 is the only possible redundant power supply setting for Solace PubSub+ 3530s and 3560s.
