ADB Flash Memory Card Replacement

This section describes how to replace a failed Solace ADB flash memory card in an ADB-000000-01 with a new Solace-provided flash memory card.

A flash memory card can only be replaced on ADB- 000000-01. Further, only install a Solace-provided flash memory card into ADB-000000-01—other memory cards are not supported. On ADB-000000-02 and ADB-04210M-01-A, the flash memory card is internal to the blade and is not accessible nor removable.

The figure below shows the faceplate of an ADB-000000-01. The ADB faceplate is accessible at the rear of the appliance chassis.

Flash Memory Card Slot and Status LED on ADB-000000-01

In the ADB faceplate there is a slot where the flash memory card used by the ADB is installed. The ADB faceplate also features a red LED that indicates the read/write status for the Solace-provided flash memory card as reported by the ADB. The following states are possible:

  • Off: No reading or writing of spooled messages from the flash memory card.
  • On: Initialization, read, or write operations are occurring for the flash memory card.
  • Blinking: The Solace provided flash memory card is not present, or is incorrectly installed.

ESD/Safety Precautions

Before replacing ADB flash memory cards, review the following safety warning.

To prevent equipment damage from Electrostatic Discharge (ESD), always ensure that the appliance chassis is electrically connected to earth ground. Always follow ESD prevention procedures when removing and replacing power supplies. Use an anti-static wrist strap, or another anti-static device. If no wrist strap or mat is available, ground yourself by touching the metal part of the chassis.

Tools and Equipment Requirements

  • Electrostatic discharge wrist strap
  • Anti-static mat
  • Replacement flash memory card (supplied by Solace)

Removing a Failed Flash Memory Card

To remove a failed flash memory card from an ADB-000000-01, use the following procedure:

  1. Use an anti-static wrist strap or other device to ground yourself.
  2. When the flash memory card LED is in an OFF state, press the flash memory card that is currently installed in the ADB to eject it from the flash memory card slot.

Installing a New Flash Memory Card

To install a Solace-provided replacement flash memory card, use the following procedure:

  1. Use an anti-static wrist strap or other device to ground yourself.
  2. When the flash memory card LED is in a Blinking state (no flash memory card is present), slide the Solace-provided flash memory card with the printed label facing right into the slot on the ADB faceplate, and gently press until it is firmly seated.
  3. The red flash memory card LED should turn On, to indicate that data being read or written to the flash memory card, or be Blank, to indicate that the flash memory card is successfully installed, but data is not being read or written to the card.
  4. If the LED is in a Blinking state, the flash memory card is not readable (for example, it might not be properly seated).

Only install a replacement flash memory card that Solace has provided for the specific ADB and Solace PubSub+ version used by your appliance.
Do not attempt to install a flash memory card not supplied by Solace or a Solace provided flash memory card in the ADB of another appliance that uses an earlier version of Solace PubSub+. Installing an incompatible flash memory card can result in undefined and inconsistent event broker behavior.
