Solace PubSub+ 3230

This section provides information that trained service technicians need to install Solace PubSub+ 3230s (CHS-3230AC-02) in a rack. It includes specific details on equipment handling and safety guidelines, site preparation, as well as the instructions, required tools and parts needed to ensure a successful installation of a Solace PubSub+ 3230.

Procedures for performing a basic configuration on Solace PubSub+ 3230s once installation is complete are provided in Initializing Appliances.
