Management Interface Port Specifications

The management network interfaces (that is, eth0 and eth1) found on the rear of the Solace PubSub+ 3230 use single RJ-45, Automatic Medium-Dependent Interface Crossover (Auto-MDIX) ports with auto-negotiation for 10/100/1000 base T.

Connect to these ports with CAT-5e unshielded twisted pair cable.

Refer to Connections, LEDs, and Specifications for the rear panel view and connections of the Solace PubSub+ 3230.
