Setting up the PubSub+ Appliance

The purpose of this section is to show you how to get a Solace PubSub+ appliance operational and ready for messaging after it has been installed and powered up using the instructions in Solace Appliance Hardware Installation. The following tasks guide you through the set up of the minimum configuration necessary to get clients connected and exchanging messages,

  • Connecting Management Consoles
  • Initial Set Up
  • Setting Up Basic Configurations

When finished, you can then remotely log on to the appliance through a Secure Shell (SSH) connection to the management interface, as well as ping the message backbone interface.

It should be noted that the information provided in this section only covers the deployment of a single appliance and does not take high-availability (HA) redundancy into consideration.
