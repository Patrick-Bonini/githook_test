Prerequisites & Installation

This section provides information for trained service technicians to install a Solace PubSub+ appliance in a rack.

Prerequisites

You must have two IP addresses available for use with a Solace PubSub+ appliance:

  • One for the Management VRF to enable out-of-band management and monitoring
  • One for the Message Backbone VRF to enable applications to establish client connections to the appliance for sending and receiving data
