SolAdmin

SolAdmin is a management application for event brokers that can be used with a user account. While the Command Line Interface (CLI) provides access to all management features for event brokers, SolAdmin provides a simplified graphical interface for managing event broker properties.

As a Java-based desktop application, SolAdmin is supported on systems running Windows, Linux, and macOS that are in active support and also have security update support.

Customer documentation for SolAdmin is provided with the application, and it can be accessed through context-sensitive help links in the user interface when using SolAdmin.

Features

Depending on the access level that is assigned to the user account you use, some of the management tasks that you can do with SolAdmin include:

  • Setting Up the SolAdmin Management Domain
  • Configuring User Authentication and Authorization
  • Performing Common Event Broker Configuration Tasks
  • Configuring Message VPNs
  • Configuring Message VPN Bridges
  • Configuring ACL Profiles
  • Configuring Clients
  • Working with Endpoints
  • Configuring JMS Objects
  • Configuring Event Broker Redundancy
  • Configuring Site Replication
  • Configuring PubSub+ Cache
  • Monitoring Event Brokers
