Authentication Methods

Users connect to a Solace PubSub+ event broker through a port enabled by the event broker. When the event broker accepts a connection from a user, the connection enters an unauthenticated state. No user requests are processed until the connection is authenticated.

A Solace PubSub+ event broker supports the following user authentication methods:

  • Authentication through an internal event broker database.
  • Authentication through an external Remote Authentication Dial-in User Service (RADIUS) server.
  • Authentication through an external Lightweight Directory Access Protocol (LDAP) server. Both LDAP and LDAP over TLS are supported.
  • Authentication through an OAuth provider.
