Default Solace Geneos Agent Data Views

The Solace Geneos Agent provides the following pre-defined data views:

  • ApplianceEnvironment—displays appliance environment information, such as mainboard temperature, fan speed and NAB temperature.
  • ADBStatus—displays the operational status of an appliance's Assured Delivery Blade (ADB).
  • HBAStatus—displays the operational status of an appliance's Host Bus Adapter Blade (HBA).
  • GlobalClientStats—displays top-level statistics about messages received from and sent to clients connected to the event broker.
  • HardwareStatus—displays the appliance's power redundancy configuration, number of operational power supplies, and slot contents.
  • MessageSpool—displays summary information about the state and current usage of an event broker's message spool.
  • MessageVPNBridge—displays information about Message VPN bridges currently configured on the event broker.
  • MessageVPNBridgeStats—displays statistics for messages sent and received over Message VPN bridges.
  • MessageVPN—displays information about Message VPNs currently configured on the event broker.
  • MessageVPNStats—displays statistics of Message VPNs on the event broker.
  • NABStatus—displays the operational status of an appliance's Network Acceleration Blade (NAB).
  • NeighborLink—displays information about CSPF neighbor links between the event broker and its neighbors.
  • NeighborStats—displays statistics about CSPF neighbor links between the event broker and its neighbors.
  • Queues—displays information about all durable message queues configured on the event broker.
  • RedundancyStatus—displays top level information regarding the redundancy configuration and status of the event broker.
  • Services—displays the status of the services on the event broker.
  • TopicEndpoints—displays information about all durable topic endpoints currently configured on the event broker.
  • CacheStats—displays information about the PubSub+ Cache instances connected to the event broker. (This monitor can only be used for appliances if PubSub+ Cache is unlocked with a product key.)
  • MonitorStatus—displays the health of different monitors by collecting gathering state and statistics, such as monitor status, missed intervals, collection and reporting time for each data view. (autostart=false in the configuration file—it must be enabled to appear.)
  • ServiceStatus—displays the health of Appliance Service, Netprobe Services and SolGeneosAgent Service, such as service state, up time, and so on. (autostart=false in the configuration file—it must be enabled to appear.)
