Removing Replication on Newly Migrated

After you have verified that your client applications are performing and functioning correctly on the Target you've migrated to, you can remove replication feature because it is no longer required.

To remove the replication feature, use the following CONFIG commands in the :

SolaceCLI (configure)# message-vpn <Message VPN name>
SolaceCLI (configure)# replication
SolaceCLI(configure/message-vpn/replication)# shutdown

where:

  • <Message VPN name> is the Message VPN that is the same between your replication pair.

recommends that after you remove replication you fully test that your client applications function as expected.
