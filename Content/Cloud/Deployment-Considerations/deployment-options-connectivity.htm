Private IP Addresses

Private IP addresses connect messaging applications to s over private routes inside your (customer) network. Depending on where the applications reside, it may be necessary to enable network peering between the client application VPC/VNet and the VPC/VNet where the resides. Network peering is an interconnection of administratively separate networks to allow them to exchange traffic.
