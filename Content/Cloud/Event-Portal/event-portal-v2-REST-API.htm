v2 REST API

The second generation of uses the second version of the REST API. You can use this new API to integrate your other applications and systems with and retrieve your event-driven architecture (EDA) data.

To perform REST API calls, you will need an API token. Use your account to generate an API token. For more information, see Creating an API Token.

The v2 REST API has an API reference to help you navigate your way through the REST API. You can view and download the Open API specifications.
