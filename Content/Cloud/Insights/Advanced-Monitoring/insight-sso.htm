Single Sign-On for Datadog Accounts

Single Sign-On for the Datadog account included in a subscription is a Controlled Availability feature. Contact to see if this feature is suitable for your use case and deployment requirements.

If you have a subscription, you can configure single sign-on (SSO) for the Datadog account that is included with . SSO provides users who have access to the Datadog account with permissions that are mapped to existing user roles. offers this feature as controlled availability.

When configuring SSO for the Datadog account included with your subscription, you must be aware of the following:

  • You require an existing account with a SAML Identity Provider (IdP).

  • The person you select as the as the SAML manager is responsible for configuring SSO for the Datadog account included with .

  • User permissions must be mapped only to the and roles.

  • You are responsible for managing the expiry of your SAML metadata file and must request that update the file at least one month before its expiry date.
