The Manager role is a Controlled Availability feature. Contact to see if this feature is suitable for your use case and deployment requirements.

If you have a subscription, you can assign the role to an individual in your organization to allow them to manage Datadog API and APP keys. The can also delegate access for Datadog integrations for the Datadog account included with . provides this feature as controlled availability.

When using the role, be aware of the following restrictions:

  • Only use integrations approved by (e.g., ServiceNow, PagerDuty).
