Viewing Information and Details of an Event Mesh

You can view the details of an event mesh that's been created after you've created it in . To view the details of an event mesh, you require the correct role assigned to your user profile. For more information about roles required, see Considerations for Working with Event Meshes in [%=Product-Names.cloud_product_short%].

To view information or details about an event mesh, you can perform the following tasks:

  • Check the progress of an event mesh when you create, update, or delete it. For more information, see Check Progress of an Event Mesh.
  • Discover the event meshes are in your account. For more information, see Scanning and Refreshing for New Event Meshes.
  • Refresh the view of an event mesh. For more information, see Refreshing for Changes in an Existing Event Mesh.
  • View the details of an event mesh and the status of a Health Check. For more information, see Viewing the Event Mesh and Service Details.
  • Interact with the map to get additional information. For more information, see Advanced Interaction with Event Mesh from the Map View.

Before view an event mesh, we recommend that you review the considerations. For this information, see Considerations for Working with Event Meshes.

Checking the Progress of an Event Mesh

The process to create, update, or delete an event mesh takes up to five minutes. The time varies based on various factors (e.g., size of the event mesh, the cloud providers and regions of your , network connectivity speeds). When you choose to create, update, or delete an event mesh, you get a brief notification on the page to indicate that the operation has started. To check on the progress of the operation, you can navigate to the event mesh and see its status. You may need to refresh the view or discover the event mesh if you've just created it. For more information, see Scanning and Refreshing for New Event Meshes.

Discover New Event Meshes

You may have manually created an event mesh or have existing event meshes that you've created that aren't shown. You can discover these meshes by performing a scan that refreshes your view of event meshes in your account (Workspace). To do this, use the following steps:

  2. Select from the navigation bar.

  3. On the Mesh Manager: Event Meshes page, click Mesh Actions beside the Create Mesh link and select Scan for Event Meshes.

After a few minutes, the cards representing the event meshes refresh. If an event mesh is in the process of being created, updated, or deleted, you can see its progress. For more information, see Check Progress of an Event Mesh.

Refreshing the View for an Event Mesh

After you create, add, or remove links from an event mesh, you can refresh the visual representation for the event mesh. To do so, you can run a scan to refresh your view of your event mesh to see any link or topology changes to your event mesh. Only a single scan can occur at one time. To refresh the view for your event mesh, use these steps:

  2. Select from the navigation bar.

  3. On the Mesh Manager: Event Meshes page, on the card for the event mesh your want to refresh, click Mesh Actions, and in the menu, select one ways to refresh the links:

    • Scan Event Mesh to refresh the links on the

    • View Details and then on the Mesh Map page, click Mesh Actions, and then click Scan Event Mesh

As the scan occurs, updates to the visual representation of your mesh appear. A Scan completed notification pop-up notification appears on the bottom of the page and you may need to refresh your web page to see the changes reflected in the user interface.
