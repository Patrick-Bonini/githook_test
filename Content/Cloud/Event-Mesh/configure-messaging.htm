Configuring and Managing Messaging

s provide the infrastructure to transmit event messages between publishers and subscribers in an event-driven architecture (EDA). Each must be configured to allow published messages to be delivered to subscribers. For general information about how event brokers work in your EDA, see What is Messaging?

This section includes information to help you configure, manage, and track messaging on your s. It includes the following topics:

  • Configuring Partitioned Queues
  • Configuring Topic Endpoints
  • Configuring Message Replay
  • Configuring Kafka Bridging
  • Creating a Message VPN Bridge
  • Enabling Distributed Tracing

For more in-depth information about configuring messaging, see Sending and Receiving Messages.
