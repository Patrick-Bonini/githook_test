Using the Learning Center

The Learning Center gives access to additional learning resources to help you become more familiar with .

The Acme Retail sample in the Learning Center includes sample environments, application domains, and model event meshes with runtime data that can simulate importing topics and consumer groups from a runtime Kafka broker.

For more information, see the following sections:

  • Using the Sample Data

Using the Sample Data

The Learning Center steps you through locating and using the sample data in . You must be an Administrator to load the sample data into . After the sample is loaded, s can access it.

If you want to find the data yourself after the sample is loaded, the sample environments, application domains, and modeled event meshes all begin with (Sample).

To locate the sample data in , use filters to show only objects in the sample application domains. For more information see Searching for Objects.

Importing Objects from Sample Audit Results

In , you can try out importing sample data. For more information on importing audit results, see Auditing and Importing Runtime Data.

The model event brokers in the sample were created to provide an example of how to use the audit and import features. They are not connected to actual s or runtime event brokers.

When you import sample data from the Audit tab, you can import it into any application domain, not just the sample application domains.

Deleting the Sample Data

When users in your organization are done exploring the sample, you can delete all the sample sample environments, application domains, and modeled event meshes from .

Objects imported from the sample audit results into application domains other than the sample application domains are not deleted. The sample can't be deleted if a user has created a reference between a sample object and an object outside of the sample.

To delete the sample, perform these steps:

  1. In the Learning Center click More Actions for the sample.
  2. Select Delete Sample.
