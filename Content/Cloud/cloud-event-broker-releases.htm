Software Event Broker Versions and Support in

This section shows the support dates, when upgrades to a version are available, and planned releases of versions in . If you want an understanding of the differences, see Types of Releases in [%=Product-Names.cloud_product_short%]. All s are deployed on Kubernetes infrastructure.

The support time frames for the versions in are different from the s releases. The support dates, upgrade availability dates (if supported), and planned releases are summarized on the Support Dates for Release Versions page. However, if you want an understanding of the differences, see Types of Releases in [%=Product-Names.cloud_product_short%].
