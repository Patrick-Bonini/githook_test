Using Deletion Protection

You can enable deletion protection to prevent an event broker service from being deleted in . You must have the or Administrator role to enable deletion protection.

Services are not protected from deletion by default.

Enabling Deletion Protection

To enable deletion protection, perform these steps:

  2. In the , select from the navigation bar.
  3. Select the you want to protect from deletion.
  4. On the service details page for the service, select the Manage tab.

  5. Click Deletion Protection.

Disabling Deletion Protection

To disable deletion protection, perform these steps:

  2. Select from the navigation bar.
  3. Select the for which you want to disable deletion protection.
  4. On the service details page, click on the Manage tab.
  5. Click Deletion protection to disable protection.
