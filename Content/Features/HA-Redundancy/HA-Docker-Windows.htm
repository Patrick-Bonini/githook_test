HA Group Configuration for Windows

This section shows you the steps to manually set up and run an HA redundancy group of PubSub+ containers in Docker Desktop (or an equivalent desktop container environment such as Rancher Desktop) on a Windows platform.

Alternatively, if you'd like to set up an HA group with Docker Compose (or equivalent), take a look at the template and instructions at GitHub: https://github.com/SolaceLabs/solace-ha-docker-compose.
