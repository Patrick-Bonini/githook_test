Configuring Events and Thresholds

You can use the Command Line Interface (CLI) to customize the formatting of generated events and their output destinations. You can also configure the threshold values required for some types of events.

A threshold-based event is generated depending on the value of a corresponding metric (for example, the number of simultaneous client connections). There are two types of thresholds that can be configured:

  • set threshold—when the corresponding metric reaches or exceeds this value, indicating that the metric is too high, an event is generated
  • clear threshold—when the corresponding metric reaches or falls below this value, indicating that the metric is back to an acceptable operating value, the event is cleared

Thresholds can be set either as specific values or percentages, and you can specify event thresholds at various levels. You can also customize the format of events and manually clear certain events. For details, see the following:
