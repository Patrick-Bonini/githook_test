Viewing the Error Logs

You can view the error logs for a . The logs are useful to understanding the issues that occurred while deploying the and provide you information to resolve the issue. Only the last ten error logs are available.

To view the error logs for a connector flow, perform the following steps:

  2. On the navigation bar, select Connectors.

  3. On the Connector Flows page, click the name of the connector flow you want to view the error logs for.

  4. On the page, click the ellipsis and select View Error Logs.

    The View Error Logs option is available only if the has been deployed at least once..

  5. From the list of logs on the Recent Error Logs page, select the error log to view its details.
