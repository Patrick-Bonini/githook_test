Management Tools

Use these tools to manage and monitor your Solace PubSub+ event brokers:

  • Solace PubSub+ Manager—A browser-based administration console packaged with the Solace PubSub+ event broker.
  • Solace CLI—A text-based command line interface for configuring and monitoring.
  • Solace Element Management Protocol (SEMP)—A RESTful API that can be used to configure Solace PubSub+ event brokers.
  • SolAdmin—A management application that can be used with a CLI user account.
  • Solace PubSub+ Monitor—An application for monitoring your Solace PubSub+ event brokers.
  • System Resource Calculator—A tool that calculates the system resources required based on system limits.
