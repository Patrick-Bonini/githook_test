Initial Context Property

The Initial Context property can be used to specify an Initial Context Factory.

Initial Context Factory

  • Type: String
  • Format: Only com.solacesystems.jndi.SolJNDIInitialContextFactory is supported.
  • Default: None

Initial Context Factory Property
Property Source
(in descending priority)		 Example

Initial Context

env.put(InitialContext.INITIAL_CONTEXT_FACTORY, “com.solacesystems.jndi.SolJNDIInitialContextFactory”);

System

-Djava.naming.factory.initial=com.solacesystems.jndi.SolJNDIInitialContextFactory

JNDI Properties File

java.naming.factory.initial=com.solacesystems.jndi.SolJNDIInitialContextFactory
