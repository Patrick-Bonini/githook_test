Integrating JMS Without the Resource Adapter

To integrate the JMS API into a JEE environment without using the JCA Resource Adapter, you should only use the following library files from the JMS API to avoid potential conflicts with existing Java library files:

  • commons-logging-1.1.x.jar
  • sol-jms-10.x.x.jar
  • These library files are present in the lib directory of the extracted JMS package. Copy these library files to the relevant directory for your application. Then add them to your CLASSPATH to use as a JMS provider within a JEE container.
