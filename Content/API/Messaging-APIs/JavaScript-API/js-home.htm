JavaScript API

The Messaging API for JavaScript (version ) allows web browsers and mobile client applications to send and receive messages with event brokers. The JavaScript, Node.js, Python, and Go Messaging APIs from share the same asynchronous and non-blocking programming model, but are packaged differently, and optimized for their respective target environments.

Message Exchange Patterns

The Messaging API for JavaScript supports all common message exchange patterns.

  • Publish / Subscribe
  • Point-to-Point
  • Request / Reply

For more information, see Message Exchange Patterns.
