Node.js API

The Messaging API for Node.js (version ) can be used in cloud-based, enterprise scale server applications. The Node.js, JavaScript, Python, and Go Messaging APIs from share the same asynchronous and non-blocking programming model, but are packaged differently, and optimized for their respective target environments.

Message Exchange Patterns

The Messaging API for Node.js supports all common message exchange patterns.

  • Publish / Subscribe
  • Point-to-Point
  • Request / Reply

For more information, see Message Exchange Patterns.
