Java RTO API

The Messaging API for Java RTO, version (also referred to as solclientj) is a low-latency Java Native Interface (JNI) wrapper for the C API.

Message Exchange Patterns

The Messaging API for Java RTO supports all common message exchange patterns.

  • Publish / Subscribe
  • Point-to-Point
  • Request / Reply

For more information, see Message Delivery Modes.
