C# / .NET API

The Messaging API for .NET, version (also referred to as SolClient for .NET) is an object-oriented, managed wrapper for the C API.

Message Exchange Patterns

The .NET API supports all common message exchange patterns.

  • Publish / Subscribe
  • Point-to-Point
  • Request / Reply

For more information, see Message Exchange Patterns.
