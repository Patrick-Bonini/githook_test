Go API

The Messaging API for Go (version ) can be used in cloud-based and enterprise-scale server applications. You can use the PubSub+ Go API with any version of Go that is in support (see Google, Supported Go Versions). The Go, Python, Node.js, and JavaScript Messaging APIs share the same asynchronous and non-blocking programming model, but are packaged differently and optimized for their respective target environments.

Message Exchange Patterns

The Messaging API for Go supports the following message exchange patterns:

  • Publish / Subscribe
  • Request / Reply

For more information, see Message Exchange Patterns.
