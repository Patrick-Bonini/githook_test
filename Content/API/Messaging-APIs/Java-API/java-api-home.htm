Java API

The Messaging API for Java (version ) utilizes modern interfaces, patterns, and usability in a native Java environment. This Java API is suited for developers who want ease-of-use, require a powerful enterprise API, and want to use the next generation of Java APIs from . Like the classic Java API (JCSMP), this Java API delivers high message-throughput. For more information about the classic Java API, see JCSMP.

Message Exchange Patterns

The Messaging API for Java supports the following message exchange patterns:

  • Publish / Subscribe
  • Point-to-Point
  • Request / Reply

For more information, see Message Exchange Patterns.
