JMSToolBox

You can use JMSToolBox to perform a variety of tasks with PubSub+ event brokers via the JMS API. Some useful features of JMSToolBox are:

  • automatically discovering queues and topics
  • browsing and viewing the contents of messages
  • generating messages using a templating scheme
  • scripting the generation of messages

JMSToolBox is an open source project that Solace has contributed to, but does not own or directly support. For help with JMSToolBox, check the Solace Developer Community.
