Supported Environments for the

The Messaging API for Java is compatible and supported with any version of the Oracle JDK or OpenJDK that is in active support and also has security update support. For information about supported versions, see the Oracle Java SE Support Roadmap. The Java API has not been tested withÂ Android platforms.

You can retrieve the Java API using Maven (pom.xml) or Gradle (build.gradle).

Maven example:

<dependency>
  <groupId>com.solace</groupId>
  <artifactId>solace-messaging-client</artifactId>
  <version>.0</version>
</dependency>

// Solace Messaging API for Java Dependencies
