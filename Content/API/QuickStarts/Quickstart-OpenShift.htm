OpenShift Quick Start

Solace provides a Docker version of PubSub+ so you can use it in OpenShift, and developed OpenShift quick start templates to automate the deployment and configuration of PubSub+ in an OpenShift environment. In addition, we have made available a template to demonstrate how PubSub+ can be used by a Demo application in OpenShift. You can use the templates as a starting point for your own implementation.
