Azure Quick Start

Solace has developed a Microsoft Azure Quick Start template that automates the deployment of PubSub+ into any Azure region in the configuration of your choice.

With this Azure Quick Start, PubSub+ can be deployed in either a standalone configuration or in high-availability (HA) redundancy groups for fault tolerance. In the HA configuration, if one of the event brokers fails or is taken out-of-service, the other automatically takes over and provides service to the clients of the event broker that is out-of-service. To increase availability, the event brokers are deployed across three different Fault Domains.

Deployment and configuration tasks are automated by a template that you can customize during launch. You can also use the template as a starting point for your own implementation. The Quick Start includes a guide with step-by-step deployment and configuration instructions.

The template is available from the SolaceProducts GitHub project or from the Azure template library.

Kubernetes

If you want to deploy an Event Broker HA Group using Kubernetes with Azure, follow the Azure documentation to deploy an Azure Kubernetes Service (AKS) cluster, and then follow the instructions in the Kubernetes Quick Start to deploy the HA broker group.
