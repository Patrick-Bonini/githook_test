GCP Quick Start

A Quick Start template is available at GitHub that can help you to setup a Solace PubSub+ software event broker Docker Container onto a Google Compute Engine Virtual Machine CentOS host. This provides a deployment template that lets you deploy a Solace PubSub+ event broker into any region of Google Cloud Platform (GCP).

You can also deploy a software event broker HA group onto a Google Kubernetes Engine (GKE) cluster using the Quickstart template available in GitHub.
