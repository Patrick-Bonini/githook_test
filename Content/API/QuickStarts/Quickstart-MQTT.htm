MQTT Auto-Reconnect and Publish

This project provides a simple example that shows how to use MQTT Paho Java 1.1.0 features of auto-reconnect and publish while offline. These features enhance the MQTT system high availability.

This project is the base example from which interoperability with Solace PubSub+ high availability features of redundancy and disaster recovery where tested.

This project depends on maven for building. To build the class locally, check out the project and build from the source by doing the following:

git clone https://github.com/SolaceLabs/mqtt-3.1.1-test
cd /mqtt-3.1.1-test
mvn compile
     or
gradle build
