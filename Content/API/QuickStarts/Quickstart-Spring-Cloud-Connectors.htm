Spring Cloud Connectors for PubSub+

This project provides an implementation of the ServiceInfo and ServiceInfoCreator interfaces to extend the Spring Cloud Connectors project to the Solace PubSub+ Cloud Foundry service.

Using this in your Spring application can make consuming the PubSub+ messaging service simpler than straight parsing of the VCAP_SERVICES environment variable.

For an introduction to Cloud Connectors and an overview of the options for extending Spring Cloud, visit the Cloud Spring site.
