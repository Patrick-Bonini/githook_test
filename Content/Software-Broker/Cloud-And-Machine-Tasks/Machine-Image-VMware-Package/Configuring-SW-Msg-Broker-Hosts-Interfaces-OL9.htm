Software Event Broker Host Interface Configuration

The default behavior for a software event broker machine image in a hypervisor is to dynamically receive its IP address configuration from the host. Alternatively, you can also manually associate an IP interface with a host OS physical interface through the or configuration keys. However, before any host OS physical interface can be used by the event broker, it first must be configured on the host.

Configuring Interfaces with Static IP Addresses

Solace software event brokers support only a single network interface. Configuring multiple interfaces is not supported.
