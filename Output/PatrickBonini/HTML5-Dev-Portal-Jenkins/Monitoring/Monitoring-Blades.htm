Monitoring Hardware Blades

To monitor the state and performance of the hardware blades in a Solace PubSub+ appliance, enter the following User EXEC command to display the serial number, firmware versions, and error counts for the blades installed in the appliance:

solace> show hardware [details | post]

Where:

details asks to show detailed information about the appliance.

post asks to show Power-On Self Test (POST) status.
